#### Features

- Four alert types: `success`, `warning`, `danger`, `info`
- Optional title, description, and icon
- Custom action element support
- Closable via a close button
- Customizable size and style


#### Basic Info Alert

```tsx
<Alert
  title="Heads up!"
  description="This is an informational message."
  type="info"
/>
```

```tsx

  title="Session expired"
```

#### Accessibility

- ARIA `role="alert"` for important messages
- Color contrast meets WCAG 2.1 AA
- Keyboard-navigable close button

#### Best Practices
- Use `title` for critical information
- Keep `description` under 2 lines when possible
- Reserve type="danger" for actual errors
- Include actions only when immediate response is needed

#### Troubleshooting
- Alert not closing: Ensure you're either using uncontrolled mode or properly handling `onClose`
- Missing styles: Verify `QuenUIProvider` is wrapping your app
- Icon alignment: Use consistent icon sizes (recommended 16px)
