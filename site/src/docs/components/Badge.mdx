#### Accessibility
- Automatic ARIA role="status" for dynamic badges
- Color contrast meets WCAG 2.1 AA standards
- Properly scales with browser text size changes

#### Best Practices
- Use for short labels (1-3 words)
- Reserve red (`danger`) for urgent statuses
- Pair icons with text for clarity
- Maintain consistent badge usage across application
