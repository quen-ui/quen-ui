#### Default Button

```tsx
<Button onClick={() => console.log("Clicked!")}>Click Me</Button>
```

A standard button with default medium size (size="m") and primary style (view="primary"). Triggers the onClick callback when clicked.

<Button onClick={() => console.log("Clicked!")}>
  <span>Click Me</span>
</Button>

#### Disabled State

```tsx
<Button disabled onClick={() => console.log("This will not trigger")}>
  Disabled Button
</Button>
```

A non-interactive button with disabled styling. Ignores all click and keyboard events when isDisabled={true}.

<Button disabled onClick={() => console.log("This will not trigger")}>
  <span>Disabled Button</span>
</Button>

<Button

#### Loading State

```tsx
function SubmitButton() {
  const [isLoading, setLoading] = useState(false);

  const handleClick = () => {
    setLoading(true);
    // Simulate API call
    setTimeout(() => setLoading(false), 2000);
  };

  return (
    <Button loading={isLoading} onClick={handleClick}>
      {isLoading ? "Processing..." : "Submit"}
    </Button>
  );
}
```

Shows a loading spinner (Loader component) when isLoading={true}. Automatically blocks click events during loading.

<Button loading>
  <span>Processing...</span>
</Button>

#### Custom Loader

```tsx
<Button
  loading
  loaderProps={{
    view: "dots", // Loader type
    color: "white", // Spinner color
    size: "xs"
  }}>
  Saving Changes
</Button>
```

<Button
  loading
  loaderProps={{
    view: "dots", // Loader type
    color: "white", // Spinner color
    size: "xs" // Spinner size in pixels
  }}>
  <span>Saving Changes</span>
</Button>

Customize the loading indicator with loaderProps. Works with all Loader component variants (oval, dots, etc.).

#### Accessibility Notes

- Keyboard Navigation: Buttons can be activated with `Enter`/`Space` keys.
- Focus States: Visible focus rings for keyboard users.
- ARIA Attributes: Automatically adds `aria-disabled` and `aria-busy` for disabled/loading states.
