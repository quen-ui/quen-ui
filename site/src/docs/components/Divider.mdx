#### Basic horizontal divider

Creates a standard horizontal divider spanning 100% width with centered alignment. Uses the default primary view style.

```tsx
import { Divider } from "@quen-ui/components";

const SectionSeparator = () => {
  return (
    <div>
      <h2>Section 1</h2>
      <Divider direction="horizontal" />
      <h2>Section 2</h2>
    </div>
  );
};
```

<div>
  <h2>Section 1</h2>
  <Divider direction="horizontal" />
  <h2>Section 2</h2>
</div>

#### Vertical divider

```tsx
<div style={{ display: "flex", height: 100, gap: "8px" }}>
  <div>Left Content</div>
  <Divider direction="vertical" height="80%" />
  <div>Right Content</div>
</div>
```

<div style={{ display: "flex", height: 100, gap: "8px" }}>
  <div>Left Content</div>
  <Divider direction="vertical" height="80%" />
  <div>Right Content</div>
</div>

Creates a vertical separator between elements. Control height with the height prop (default 100%). Ideal for separating columns or sidebar content.

#### Custom alignment

```tsx
<div style={{ display: "flex", flexDirection: "column", gap: 24 }}>
  <Divider direction="horizontal" align="left" width="50%" />
  <Divider direction="horizontal" align="center" width="75%" />
  <Divider direction="horizontal" align="right" width="50%" />
</div>
```

<div style={{ display: "flex", flexDirection: "column", gap: 24 }}>
  <Divider direction="horizontal" align="left" width="50%" />
  <Divider direction="horizontal" align="center" width="75%" />
  <Divider direction="horizontal" align="right" width="50%" />
</div>

Shows different alignment options for horizontal dividers. Combine with width prop to create varied visual effects. Alignment is ignored for vertical dividers.

#### Custom dimensions

```tsx
<Divider direction="horizontal" width="200px" height="2px" view="warning" />
```

<Divider direction="horizontal" width="200px" height="2px" view="warning" />

Overrides default dimensions with custom values. For horizontal dividers:

- `width` controls length
- `height` controls thickness

For vertical dividers:

- `width` controls thickness
- `height` controls length

#### Best practices

##### When to use
- Between distinct content sections
- To separate list items in dense UIs
- Inside cards to organize content
- To create visual hierarchy in forms
- Between sidebar sections

##### When to avoid
- Between tightly related elements
- In place of proper spacing
- More than 2-3 consecutive dividers
- Without clear visual purpose
