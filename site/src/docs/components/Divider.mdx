#### Basic horizontal divider

A horizontal line separating content sections. By default, spans 100% width, aligned to center, and uses the `primary` style.

<div>
  <Title size="l">Section 1</Title>
  <Divider direction="horizontal" />
  <Title size="l">Section 2</Title>
</div>

```tsx
import { Divider } from "@quen-ui/components";

const SectionSeparator = () => {
  return (
    <div>
      <Title size="l">Section 1</Title>
      <Divider direction="horizontal" />
      <Title size="l">Section 2</Title>
    </div>
  );
};
```

```tsx
<div style={{ display: "flex", flexDirection: "column", gap: 16 }}>

```

Usage context:

- `primary` → Neutral separation
- `success` → Positive context (e.g. completed tasks)
- `warning` → Important notices
- `danger` → Critical / destructive context
- `disabled` → Muted or inactive content

#### Vertical divider

Creates a vertical separation between inline or flex items.

<div style={{ display: "flex", height: 100, gap: "8px" }}>
  <Text>Left Content</Text>
  <Divider direction="vertical" height="80%" />
  <Text>Right Content</Text>
</div>

```tsx
<div style={{ display: "flex", height: 100, gap: "8px" }}>
  <Text>Left Content</Text>
  <Divider direction="vertical" height="80%" />
  <Text>Right Content</Text>div>
</div>
```

Notes:

- Default height is 100%
- Width controls thickness
- Useful for sidebars, toolbars, and multi-column layouts

#### Custom alignment (horizontal only)

Control horizontal divider positioning relative to its container.

<div style={{ display: "flex", flexDirection: "column", gap: 24 }}>
  <Divider direction="horizontal" align="left" width="50%">
    Left
  </Divider>
  <Divider direction="horizontal" align="center" width="75%">
    Center
  </Divider>
  <Divider direction="horizontal" align="right" width="50%">
    Right
  </Divider>
</div>

```tsx
<div style={{ display: "flex", flexDirection: "column", gap: 24 }}>
  <Divider direction="horizontal" align="left" width="50%">
    Left
  </Divider>
  <Divider direction="horizontal" align="center" width="75%">
    Center
  </Divider>
  <Divider direction="horizontal" align="right" width="50%">
    Right
  </Divider>
</div>
```

Alignment does not apply to vertical dividers.

#### Custom dimensions

Change width and thickness for finer control.

<Divider direction="horizontal" width="200px" height="2px" view="warning" />

```tsx
<Divider direction="horizontal" width="200px" height="2px" view="warning" />
```

Overrides default dimensions with custom values. For horizontal dividers:

- `width` controls length
- `height` controls thickness

For vertical dividers:

- `width` controls thickness
- `height` controls length

#### In lists and cards

Use dividers to visually separate list items, card sections, or form parts.

  <Title size="s">Team Members</Title>
  <Divider direction="horizontal" />
  <ul>
    <Text as="li">John Doe</Text>
    <Divider direction="horizontal" />
    <Text as="li">Jane Smith</Text>
    <Divider direction="horizontal" view="success" />
    <Text as="li">Mike Johnson</Text>
  </ul>
</div>

Practical usage in content containers. Use dividers to separate:

- Headers from content
- List items
- Form sections
- Card elements

#### Best practices

##### When to use

- Between distinct sections
- To separate dense list items
- Inside cards and panels
- In forms to group related fields
- In toolbars and sidebars

##### When to avoid

- Between tightly coupled elements
- As a replacement for proper spacing
- More than 2–3 consecutive dividers
- Without clear visual purpose
