#### Basic settings drawer

A standard right-side drawer. Controlled by the isOpen state and closed via backdrop click, close button, or the `Escape` key.
Best suited for user settings or contextual panels.

<SettingsPanel />

```tsx
const SettingsPanel = () => {
  const [isOpen, setIsOpen] = useState(false);

  return (
    <>
      <Button onClick={() => setIsOpen(true)}>Open Settings</Button>

      <Drawer
        isOpen={isOpen}
        onClose={() => setIsOpen(false)}
        position="right"
        size="m"
        title="User Settings">
        Settings Form
      </Drawer>
    </>
  );
};
```

#### Mobile bottom drawer

A `full-screen drawer` that slides up from the bottom.
Optimized for mobile UIs and commonly used for navigation menus. Dismissal with a swipe-down gesture is recommended.

<MobileMenu />

```tsx
const MobileMenu = () => {
  const [isMenuOpen, setMenuOpen] = useState(false);

  return (
    <>
      <Button onClick={() => setMenuOpen(true)}>Open Navigation</Button>
      <Drawer
        isOpen={isMenuOpen}
        onClose={() => setMenuOpen(false)}
        position="bottom"
        size="full"
        title="Navigation"
        closeIcon={<IconXboxX />}>
        MobileNavigation
      </Drawer>
    </>
  );
};
```

#### Left-aligned navigation

A left-side navigation drawer.
The `noCloseOnClickOutside` prop prevents accidental closure, making it suitable for persistent navigation menus in desktop layout

<LeftAlign />

```tsx
const LeftAlign = () => {
  const [isNavOpen, setNavOpen] = useState(false);

  return (
    <>
      <Button onClick={() => setNavOpen(true)}>Open Navigation</Button>
      <Drawer
        isOpen={isNavOpen}
        onClose={() => setNavOpen(false)}
        position="left"
        size="s"
        title={
          <div className="logo-header">
            <span>Navigation</span>
          </div>
        }
        noCloseOnClickOutside>
        Menu
      </Drawer>
    </>
  );
};
```

#### Top notification panel

A temporary notification drawer sliding down from the top.
The increased `zIndex` ensures visibility above modals. Useful for alerts and notification centers.

<TopNotifications />

```tsx
const TopNotifications = () => {
  const [isOpen, setOpen] = useState(false);

  return (
    <>
      <Button onClick={() => setOpen(true)}>Open Notifications</Button>
      <Drawer
        isOpen={isOpen}
        onClose={() => setOpen(false)}
        position="top"
        size="s"
        zIndex={200}>
        <Flex gap={8} direction="column">
          <Alert
            title="Heads up!"
            description="This is an informational message."
            type="info"
          />
          <Alert
            title="Storage almost full"
            description="You are using 95% of your storage capacity."
            type="warning"
            isClosable
            action={<Button size="s">Upgrade Plan</Button>}
          />
        </Flex>
      </Drawer>
    </>
  );
};
```

#### Custom close handling

A drawer with a custom close control.
Instead of an icon, a button with text and an arrow is used. Ideal for shopping carts or flows where closing should be more intentional.

<CustomClose />

```tsx
const CustomClose = () => {
  const [isOpen, setOpen] = useState(false);

  return (
    <>
      <Button onClick={() => setOpen(true)}>Open Navigation</Button>
      <Drawer
        isOpen={isOpen}
        onClose={() => setOpen(false)}
        position="left"
        size="s"
        closeIcon={
          <Button view="ghost" size="s">
            <IconArrowLeft />
            Continue Shopping
          </Button>
        }
        noCloseOnClickOutside>
        <CheckboxGroup
          options={[
            { label: "option 1", value: "1" },
            { label: "option 2", value: "2" }
          ]}
        />
      </Drawer>
    </>
  );
};
```

#### Sizing comparison

Drawers support predefined sizes: mini(`xs`) small (`s`), medium (`m`), large (`l`), and full-screen (full).
Size affects width for horizontal drawers and height for vertical drawers.

<Sizing />

```tsx
const Sizing = () => {
  const [currentSize, setSize] = useState<IDrawerProps["size"]>();

  return (
    <div>
      <Flex gap={16}>
        <Button onClick={() => setSize("xs")}>XS</Button>
        <Button onClick={() => setSize("s")}>S</Button>
        <Button onClick={() => setSize("m")}>M</Button>
        <Button onClick={() => setSize("l")}>L</Button>
      </Flex>
      <Drawer
        isOpen={!!currentSize}
        onClose={() => setSize(undefined)}
        size={currentSize}
        title={`${currentSize} Drawer`}
      >
        <p>Current size: {currentSize}</p>
      </Drawer>
    </div>
  );
};
```

#### Best Practices

##### When to use
  - Secondary navigation menus
  - Contextual settings panels
  - Mobile-first interfaces
  - Multi-step forms
  - Detail views for list items
  - Notification centers

##### When to avoid
  - Critical alerts (use Modal instead)
  - Primary navigation on desktop
  - Simple tooltips
  - Actions requiring immediate attention
