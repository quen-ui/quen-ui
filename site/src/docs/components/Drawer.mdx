#### Basic settings drawer

```tsx
const SettingsPanel = () => {
  const [isOpen, setIsOpen] = useState(false);

  return (
    <>
      <Button onClick={() => setIsOpen(true)}>Open Settings</Button>

      <Drawer
        isOpen={isOpen}
        onClose={() => setIsOpen(false)}
        position="right"
        size="m"
        title="User Settings">
        Settings Form
      </Drawer>
    </>
  );
};
```

<SettingsPanel />
<br />

Standard right-side drawer for settings. Opens with button click, closes via backdrop click, close button, or Escape key.

#### Custom close handling

```tsx
const CustomClose = () => {
  const [isOpen, setOpen] = useState(false);

  return (
    <>
      <Button onClick={() => setOpen(true)}>Open Navigation</Button>
      <Drawer
        isOpen={isOpen}
        onClose={() => setOpen(false)}
        position="left"
        size="s"
        closeIcon={
          <Button view="ghost" size="s">
            <IconArrowLeft />
            Continue Shopping
          </Button>
        }
        noCloseOnClickOutside>
        <CheckboxGroup
          options={[
            { label: "option 1", value: "1" },
            { label: "option 2", value: "2" }
          ]}
        />
      </Drawer>
    </>
  );
};
```

<CustomClose />
<br/>

Customized close button with text. Prevents closing when clicking outside to avoid accidental cart abandonment.


#### Sizing comparison

```tsx
const Sizing = () => {
  const [currentSize, setSize] = useState<IDrawerProps["size"]>();

  return (
    <div>
      <Flex gap={16}>
        <Button onClick={() => setSize('s')}>Small</Button>
        <Button onClick={() => setSize('m')}>Medium</Button>
        <Button onClick={() => setSize('l')}>Large</Button>
      </Flex>
      <Drawer
        isOpen={!!currentSize}
        onClose={() => setSize(undefined)}
        size={currentSize}
        title={`${currentSize} Drawer`}
      >
        <p>Current size: {currentSize}</p>
      </Drawer>
    </div>
  );
};
```

<Sizing />
<br/>

Demonstrates different size presets. Size affects both width (horizontal) and height (vertical).


#### Accessibility features
- Focus Management: Traps focus inside drawer
- Keyboard Navigation:
  - Esc closes drawer
  - Tab cycles through focusable elements
- ARIA Attributes:
  - `role="dialog"`
  - `aria-modal="true"`
  - `aria-labelledby` for title
- Screen Reader Support: Announces open/close state
- Backdrop: Darkens main content to indicate modal state

#### Best Practices

##### When to use
- Secondary navigation menus
- Contextual settings panels
 - Mobile-first interfaces
- Multi-step forms
- Detail views for list items
- Notification centers

##### When to avoid
- Critical alerts (use Modal instead)
- Primary navigation on desktop
- Simple tooltips
- Actions requiring immediate attention
