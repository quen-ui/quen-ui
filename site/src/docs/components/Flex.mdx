import { Flex, Button, TextField } from "@quen-ui/components";

#### Basic centered layout

Creates a horizontally centered row of buttons with equal spacing (`gap="m"`) from the theme spacing scale.

<Flex align="center" justify="center" gap="m">
  <Button>Button</Button>
  <Button size="l">Button</Button>
  <Button>Button</Button>
</Flex>

```tsx

</Flex>
```

#### Form layout with consistent spacing

Builds a vertical form layout with consistent row spacing `(rowGap="l")` and right-aligned action buttons in the footer.

<Flex direction="column" rowGap="l">
  <TextField label="Name" />
  <TextField label="Email" />
  <Flex gap="m" justify="flex-end">
    <Button view="ghost">Cancel</Button>
    <Button>Submit</Button>
  </Flex>
</Flex>


#### Best practices

- Consistent Gutters: Always use theme spacing tokens (`gap="m"`, `rowGap="l"`) for consistent layout rhythm across the app.
- Vertical Rhythm: Prefer `rowGap` in vertical (`column`) layouts to create consistent vertical spacing.
- Nested Spacing: Use smaller gaps in nested containers to avoid excessive whitespace.
- Responsive Adjustments: Apply wrapping (`wrap="wrap"`) or reduce gaps on smaller screens to keep layouts compact.
