```tsx
  width={300}
```

#### Responsive image

Stretches the image to fit the container’s width while maintaining its aspect ratio by setting height="auto". Useful for banners and full-width images.

<Image
  src="https://ix-marketing.imgix.net/autotagging.png?auto=format,compress&w=1946"
  alt="cactus"
  width="100%"
  height="auto"
  className="responsive-banner"
/>

```tsx
<Image
  src="/banner.jpg"
  alt="Summer sale banner"
  width="100%"
  height="auto"
  className="responsive-banner"
/>
```

#### Image with placeholder and fallback

Shows a custom placeholder (skeleton loader) while the image is loading. If the primary source fails, a fallback image is rendered instead. This is particularly useful for avatars or API-driven images.

<Image
  src="https://ix-marketing.imgix.net/autotagging.png?auto=format,compress&w=1946"
  alt="cactus"
  fallback="/default-avatar.jpg"
  placeholder={
    <div className="skeleton-loader" style={{ width: 120, height: 120 }} />
  }
  width={120}
  height={120}
/>

```tsx
<Image
```

####  Error handling with state

Demonstrates advanced error handling using React state. Logs load errors and swaps to a fallback image when the primary source fails.

```tsx
const SafeImage = () => {
  const [error, setError] = useState(false);

  return (
    <Image
      src={error ? '' : '/unreliable-source.jpg'}
      alt="Important content"
      fallback="/fallback.jpg"
      onError={() => {
        console.error('Image failed to load');
        setError(true);
      }}
    />
  );
}
```

#### Best practices

  - Use descriptive alt text for meaningful content images.
  - Leave `alt=""` for decorative images to hide them from screen readers.
  - Optimize images with modern formats like WebP, and provide JPEG or PNG fallbacks if needed.
  - Compress assets to balance quality and performance (70–85% quality recommended).
  - Leverage CDN for faster global image delivery.
  - Apply lazy loading for images below the fold to improve page performance.
  - Test responsiveness by using `%` or `auto` dimensions where appropriate.
