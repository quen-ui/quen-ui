#### Best practices

##### When to use
- Initial page load - Display during application bootstrap
- Data fetching - Indicate async operations (API calls, DB queries)
- Form submissions - Show processing state after button click
- Content transitions - During route changes or tab switching
- Background processes - For long-running operations (exports, calculations)

##### When to not use
- Instant actions - For operations under 300ms duration
- As decorative element - When not representing actual loading
- Static content - As permanent UI element without dynamic state
