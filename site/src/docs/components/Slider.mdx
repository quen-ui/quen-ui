#### Features

- Selecting a single value or range
- Horizontal or vertical orientation
- Precise positioning with snap-to-step settings
- Clicking on a track and marks
- Draggable track â€” dragging a selected portion of a range
- Labels/marks placed directly on the line
- Tooltip for displaying current values

#### Base usage

Used for situations where you need to select a single value - volume, brightness, scale, etc.

<Slider defaultValue={30} />

```tsx
<Slider defaultValue={30} />
```

#### Slider with values, steps, and labels

Labels are useful for marking categories, levels, or specific values that the user can quickly navigate to.

<Slider
  value={40}
  step={10}
  marks={[
    { value: 0, label: "0%" },
    { value: 50, label: "50%" },
    { value: 100, label: "100%" }
  ]}
/>

```tsx
<Slider
  value={40}
  step={10}
  marks={[
    { value: 0, label: "0%" },
    { value: 50, label: "50%" },
    { value: 100, label: "100%" }
  ]}
/>
```

#### Range mode (range selection)


#### Best practice

  - Use range when you need to select an interval rather than a number.
  - Include a tooltip for interactive UIs or complex forms.
  - Change the color to match the slider's theme.
  - Enable draggableTrack for a UX similar to music and video editors.
  - Use marks to display categories or key points.
  - Remember to set a fixed height for vertical orientation.
