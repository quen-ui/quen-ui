#### Usage examples

##### Basic tabs

```tsx
<Tabs defaultValue="dashboard">
  <Tabs.TabsList>
    <Tabs.Tab value="dashboard">Dashboard</Tabs.Tab>
    <Tabs.Tab value="analytics">Analytics</Tabs.Tab>
  </Tabs.TabsList>

  <Tabs.TabPanel value="dashboard">
    <Title>Dashboard</Title>
  </Tabs.TabPanel>

  <Tabs.TabPanel value="analytics">
    <Title>Analytics</Title>
  </Tabs.TabPanel>
</Tabs>
```

#### Best practices

##### 1. Tab naming conventions
- Use short, clear labels (1-2 words)
- Match `value` props exactly between `<Tab>` and `<TabPanel>`
- Avoid special characters in `value`

##### 2. Performance optimization

```tsx
// Good - lazy loads heavy content
<Tabs.TabPanel value="reports">
  {activeTab === "reports" && <ReportGenerator />}
</Tabs.TabPanel>

// Better - use keepMounted strategically
<Tabs keepMounted={shouldPreserveState}>
```

##### 3. Keyboard navigation

- `Tab/Shift+Tab`: Focus tabs
- `Enter/Space`: Select focused tab
