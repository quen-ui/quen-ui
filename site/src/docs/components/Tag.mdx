#### Best practices

##### 1. Accessibility implementation

```tsx
// Dismissible tag with proper labels
<Tag
  isClosable
  onClickClose={handleClose}
  aria-label={`Remove ${label} filter`}
>
  {label}
</Tag>
```

##### 2. Interactive states
- Use `onClick` for selectable tags (e.g., filters)
- Use `isClosable` + `onClickClose` for removable tags
- Combine with `isDisabled` for locked states
