#### Controlled visibility

Tooltip visibility can be managed externally with the `open` prop.

<Flex justify="center">
  <TooltipControlledVisibility />
</Flex>

```tsx
const [showTooltip, setShowTooltip] = useState(true);

return (
  <Tooltip text="Custom controlled tooltip" open={showTooltip} position="top">
    <Button
      onMouseEnter={() => setShowTooltip(true)}
      onMouseLeave={() => setShowTooltip(false)}>
      Hover me
    </Button>
  </Tooltip>
);
```

#### Best practices

##### 1. Content Guidelines

- Keep tooltip text short (1â€“2 lines).
- Use for supplementary information, not primary.
- Avoid critical content inside tooltips.

##### 2. Positioning strategy

```tsx
// Default for most cases
<Tooltip position="top" />

// When space constrained
<Tooltip position={isMobile ? "bottom" : "right"} />

// For edge-aligned elements
<Tooltip position="bottomRight" />
```

###### 3. Performance optimization

- Use `open` for controlled tooltips (e.g., form validation).
- Avoid passing heavy React trees into `text`.
- Memoize complex content to reduce re-renders.
