description: "useControllableState is a React hook that implements a unified pattern for handling both controlled and uncontrolled component states. It enables components to accept a value prop (controlled mode), or maintain internal state via defaultValue (uncontrolled mode), while ensuring consistent updates through onChange. This hook is useful for any interactive component — inputs, toggles, sliders, popovers — where users may choose between controlled and uncontrolled patterns."

#### Basic usage

```tsx
import { useControllableState } from "./useMediaQuery";

function Example() {
  const [val, setVal] = useControllableState<number>({
    defaultValue: 10,
    onChange: (v) => console.log("Changed:", v)
  });
}
```

#### API

```ts
useControllableState<T>({
  value?: T;
  defaultValue?: T;
  onChange?: (v: T) => void;
})
```
