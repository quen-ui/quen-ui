This hook is ideal for optimizing performance in high-frequency events like search inputs, window resizes, or API calls triggered by user interactions."

#### Basic usage

```tsx
import { useDebounce } from "@quen-ui/hooks";

function SearchExample() {
  const handleSearch = (query: string) => {
    console.log("Executing search for:", query);
    // Perform API call or heavy computation here
  };

  const debouncedSearch = useDebounce(handleSearch, 300);

  return (
    <input
      type="text"
      placeholder="Type to search..."
      onChange={(e) => debouncedSearch(e.target.value)}
    />
  );
}
```

#### API

```ts
useDebounce<T extends (...args: any[]) => any>(
  callback: T,
  delay: number
): T
```
