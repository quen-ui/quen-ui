#### Basic usage

A simple pagination example with default settings.

```tsx
import { usePagination } from "@quen-ui/hooks";

function Example() {
  const { range, currentPage, nextPage, previousPage } = usePagination({
    total: 100,
    pageSize: 10,
  });

  return (
    <div>
      <p>Current page: {currentPage}</p>
      <div>
        <button onClick={previousPage}>Previous</button>
        {range.map((item, index) =>
          item === "dots" ? (
            <span key={index}>...</span>
          ) : (
            <button key={index}>{item}</button>
          )
        )}
        <button onClick={nextPage}>Next</button>
      </div>
    </div>
  );
}

```

#### With controlled page change

Using the `onChange` callback to react whenever the current page changes.

```ts
const { currentPage, setPage } = usePagination({
  total: 50,
  pageSize: 5,
  onChange: (page) => console.log("Page changed:", page),
});

// Manually jump to page 3
setPage(3);
```

#### Custom boundaries and siblings

Controlling how many boundary pages and sibling pages are shown.

```ts
const { range } = usePagination({
  total: 200,
  pageSize: 10,
  boundaries: 2,
  siblings: 2,
});

// Example range: [1, 2, "dots", 5, 6, 7, "dots", 19, 20]
```

#### Navigation helpers

Using built-in helpers for navigation:

```ts
const { firstPage, lastPage, nextPage, previousPage } = usePagination({
  total: 120,
  pageSize: 10,
});

firstPage();    // Go to first page
lastPage();     // Go to last page
nextPage();     // Go to next page
previousPage(); // Go to previous page
```

#### Best Practices

- Use `onChange` for data fetching – When fetching paginated data from an API, always handle page changes in the `onChange` callback.
- Set `pageSize` properly – Ensure that the number of items per page matches your backend or dataset.
- Keep boundaries small – More than 2 boundary pages may clutter the pagination UI.
- Avoid over-navigation – The hook automatically clamps page values within valid limits, but you should still handle cases where no navigation is needed.
